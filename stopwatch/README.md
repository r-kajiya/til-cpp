# Stopwatch

実行速度を計測するためのクラス

ほとんどOptimized c++からとってきた

ms単位での計測ができ、デフォルトではstd::coutに出力する

# 使い方

'
    // 計測開始

    Stopwatch stopwatch {};

    // 計測時間表示

    stopwatch.Show ();

    // 計測終了

    stopwatch.Stop ();
'
